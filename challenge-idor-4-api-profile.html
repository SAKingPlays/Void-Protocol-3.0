Security Advisory

                
The user profile API uses predictable sequential profile IDs. The endpoint does not implement authorization checks to verify if the requester has permission to view the requested profile.

            


            

                
System Notes

                
Administrative profiles may contain sensitive metadata and elevated role assignments. These profiles are typically assigned higher profile ID numbers in the sequence.
