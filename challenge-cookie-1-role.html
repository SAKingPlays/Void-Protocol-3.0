Security Advisory

                
The system determines user roles based on client-side cookie values. Role assignments are not validated server-side beyond checking the cookie value.

            


            

                
System Notes

                
Administrative functions and sensitive data are only accessible when the role cookie is set to specific values. The cookie can be modified by the client.
