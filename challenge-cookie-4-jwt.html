Security Advisory

                
The JWT token signature is not verified by the server. Only the payload is decoded and validated. This allows token tampering without detection.

            


            

                
System Notes

                
Modify the role field in the JWT payload to "admin" and re-encode using Base64 URL-safe encoding. The signature can be any value as it is not validated.

            


            

                
JWT Structure

                

                    Header: Base64URL

                    Payload: Base64URL

                    Signature: Not validated

                    Key Field: role
