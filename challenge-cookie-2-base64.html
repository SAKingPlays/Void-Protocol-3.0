Security Advisory

                
Session data is stored in a Base64 encoded cookie. The server decodes the cookie and evaluates the content. The encoding is not signed or encrypted.

            


            

                
System Notes

                
Decode the Base64 string, modify the role field, and re-encode to escalate privileges. The server accepts any valid Base64 encoded session data.
