Security Advisory

                
Tasks are organized by project name. The API accepts project names without authorization. Hidden projects exist beyond the standard alpha project. Guess project names based on system hints.

            


            

                
System Notes

                
Standard project is "alpha". Hidden projects may include "beta", "internal", "admin", and others. Internal projects may contain high-priority tasks with sensitive information.

            


            

                
Known Projects

                

                    Standard: alpha

                    Hidden: beta, internal, admin

                    Access: no authorization check
