Security Advisory

                
Session access control is based on a timestamp value. The system checks if the time value exceeds a threshold. This can be manipulated by setting a large timestamp.

            


            

                
System Notes

                
For admin access: role must be "admin" and time must be greater than 1000000. Set a large timestamp value to bypass the time-based restriction.

            


            

                
Access Requirements

                

                    role: admin

                    time: > 1000000
